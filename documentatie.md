## Setup project

```bash
 npm install
```

## Compile and run the project

```bash
# development
 npm run start:dev

# production mode
 npm run start

 # build
 npm run build
```

## Aply seed

```bash
 npm run seed
```

## Migration

```bash
# create a new migration
 npm run db:migrate:dev

# generate Prisma types
 npm run db:generate

 #  apply the migration to the database
 npm run db:migrate:run

```

## Code formatting

```bash
# prettier
npm run format

# ESLint
 npm run lint
```
